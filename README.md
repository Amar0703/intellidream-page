# 🚀 IntelliDream – Next-Gen AI Data Automation Platform

A premium, responsive SaaS landing page built for **Frontend Battle 3.0 – Phase 1** organized by **Web & Design Society, IIT Bhubaneswar**.

## 🌐 Live Demo

**Website:** https://intellidream-page.lovable.app/#GH

## 📌 Features

- ✨ Premium AI SaaS Landing Page
- 📱 Fully Responsive Design
- 🧩 Bento Grid Layout
- 📂 Mobile Accordion
- 💰 Dynamic Pricing Matrix
- 🌍 Multi-Currency Support (INR / USD / EUR)
- 🔄 Monthly & Annual Billing
- ⚡ High Performance
- 🔍 SEO Optimized
- ♿ Accessible UI
- 🎨 Modern Animations
- 🚀 Fast Loading

## 🛠 Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
- HTML5
- CSS3

## 🚀 Getting Started

```bash
git clone https://github.com/Amar0703/intellidream-page.git
cd <intellidream-page>
npm install
npm run dev
```
